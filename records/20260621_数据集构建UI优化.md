# 数据集构建 UI 优化

## 变更内容

### 1. HTML 结构修复
- 移除多余的 `</div>` 闭合标签（upload-card 和 session-card 之间），修复 mid-row 布局断裂
- HTML 标签平衡验证通过（110 开 / 110 闭）

### 2. 移除 V1/V2 用户可见标签
- 会话标题："数据集构建会话（支持手动/离线/在线三种来源）" → "数据集构建会话（手动上传 · 离线检测 · 在线检测）"
- 底部折叠区："已有数据集" → "参考音频库"，描述文字同步更新
- 移除 `.badge-v1` / `.badge-v2` CSS 类
- 清理全部注释和可见文本中的 V1/V2 引用
- JS 内部函数名称保留，不影响用户界面

### 3. 布局紧凑化与多栏对齐
- 容器宽度：`1280px` → `1400px`，与主站导航栏对齐
- 卡片 padding：`12px` → `10px`，阴影减轻
- 全局字体大小微调（分类标签 `14px→12px`，筛选标签 `13px→12px`，按钮 `14px→13px`）
- 上传区域：dropzone 图标 `48px→32px`，内边距减小
- 统计卡片：`padding` 增大至 `12px 10px`，数字字号 `26px`
- 统计行改用 CSS Grid `repeat(5, 1fr)` 5 等分（4 统计卡 + 分布图）
- 两栏比例统一：上下行均采用 `1.3:1` 比例，保证右侧列（构建会话 / 片段列表）宽度一致
- 窄屏响应式：统计行 2 列，图表全宽；中栏和内容栏纵向排列

### 4. 删除 130+ 行重复 CSS
- `.session-list` / `.session-item` 重复定义清理（原文件中出现 3 次）
- 合并精简样式表

### 5. 统计图表去外部依赖
- 移除 `chart.js` CDN 加载（不再依赖网络）
- 饼图改用纯 CSS `conic-gradient` 渲染，稳定可靠
- 数据为空时显示灰色圆环，有数据时显示三色分布
- 图表卡片与统计卡片同行显示，`flex-direction: row` 水平布局

### 6. 标注信息持久化展示
- 选中片段后在标注区上方显示文件路径、标注时间、时长
- 标注提交后在日志中输出详细标注信息（文件名、标签、时间）
- 前端标注后记录 `annotated_at` 时间戳

### 7. 确认导入功能优化
- 合并两步确认（是否全部 → 选择部分）为一步
- 导入前在日志输出配置信息（会话ID、比例、参考音频）
- 导入后在日志输出各类别划分统计
- 失败时记录错误日志

### 8. iframe 嵌入适配 (index.css)
- `.embedded-panel`：`min-height` → `height: 100%`
- 内容容器增加 `overflow: hidden` + `height: calc(100vh - 140px)`，防止双滚动条

### 9. 需求文档清理
- 移除 "未提交" 状态标记和过时的工作记录引用
