# index.html 拆分 + 任务管理 Tab 增强

- **时间**: 2026-06-20 17:00
- **任务**: 将 5799 行的 index.html 拆分为多模块文件 + 增强任务管理功能

---

## 一、拆分动机

原 `frontend/index.html` 高达 5799 行、279KB，所有 CSS（680 行）和 JS（4200 行）全部内联在单个文件中，维护极其困难。任何功能修改都需要在整个文件中定位，且文件作用边界模糊。

## 二、拆分方案

### 目录结构

```
frontend/
├── index.html              # 仅保留 HTML 骨架（982 行）
├── css/
│   └── index.css           # 全部样式（原内联 680 行）
└── js/
    ├── app.js              # 核心模块：登录/API/弹窗/Tab切换/音频播放/设备加载
    ├── offline-detection.js # 离线检测：文件上传/WebSocket/检测执行/结果展示
    ├── realtime-monitor.js  # 实时监控（当前隐藏，保留代码）
    ├── task-manager.js      # 任务管理（增强版）
    ├── reference-audio.js   # 参考音频库管理
    ├── client-monitor.js    # 客户端监控
    ├── cluster.js           # 特征聚类（当前隐藏，保留代码）
    ├── zero-shot.js         # 零样本检测（当前隐藏，保留代码）
    └── few-shot.js          # 少样本检测（当前隐藏，保留代码）
```

### 加载顺序

JS 加载顺序严格按照依赖关系：
1. `app.js` — 先定义所有全局变量、API_BASE、showModal 等基础设施
2. 各功能模块按顺序加载，均依赖 app.js

### HTML 骨架

index.html 从 5799 行 → **982 行**，减少了 83%，现在只包含：
- `<head>` 中的 meta + CSS 链接
- `<body>` 中的各 tab 的 HTML 结构
- 底部的 9 个 `<script>` 标签

## 三、路径兼容性

后端将 `frontend/` 目录挂载为 `/static`，因此引用路径均使用 `/static/` 前缀：
- `<link rel="stylesheet" href="/static/css/index.css">`
- `<script src="/static/js/app.js">`

## 四、任务管理增强

原任务管理仅有一个表格 + 刷新/清理按钮，增强内容包括：

| 功能 | 实现 |
|------|------|
| 📊 统计卡片 | 5 张卡片（总/运行中/排队/已完成/失败），调用 `/api/tasks/stats` |
| 🔍 筛选栏 | 按状态下拉、算法下拉、任务ID搜索 |
| 📈 进度条 | 表格内嵌迷你进度条，完成变绿色 |
| 📄 分页 | 上一页/下一页/首页/末页，后端已支持 limit/offset |
| 🗑️ 单任务删除 | 调用 `DELETE /api/tasks/{task_id}`，仅对终态任务可用 |
| 🔄 失败重试 | 引导用户到离线检测重试（按钮占位） |
| 📋 结构化详情 | 使用 modal 的 innerHTML 展示完整的任务信息（基本信息、进度条、文件级结果） |
| ⏱ 耗时显示 | 自动计算任务从开始到完成的耗时 |
| 🔗 统计联动 | 点击统计卡片可筛选对应状态的任务 |

### 新增 CSS 样式

在 `css/index.css` 新增的样式：
- `badge-info`, `badge-secondary`
- `filter-bar` — 筛选栏
- `progress-mini` / `progress-mini-bar` / `progress-mini-fill` — 表格内迷你进度条
- `pagination` — 分页
- `stat-card.clickable`, `stat-card.active-filter` — 统计卡片交互
- `task-detail-section`, `task-detail-grid` — 任务详情弹窗

### 新增 JS 函数

在 `task-manager.js` 中新增/重写的函数：
- `loadTaskStats()` — 加载统计信息
- `loadTaskList()` — 重写，增加筛选/搜索/分页支持
- `filterByStatus(status)` — 统计卡片点击筛选
- `applyFilters()` — 综合筛选
- `goToPage(page)` — 分页跳转
- `renderPagination()` — 渲染分页控件
- `getStatusBadge(status)` — 带图标的状态标签
- `getTaskDuration(task)` — 计算任务耗时
- `formatDuration(seconds)` — 格式化时长
- `formatTime(timeStr)` — 格式化时间
- `showTaskDetailModal(taskId)` — 结构化任务详情
- `renderTaskDetailHtml(taskId, data)` — 渲染详情 HTML
- `deleteSingleTask(taskId)` — 单任务删除
- `retryTask(taskId)` — 失败重试
